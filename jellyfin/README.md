# Jellyfin

## Purpose

Jellyfin is a self-hosted media server used to organize and stream movies, TV shows, anime, audiobooks, and photos across devices.

## Access

* URL: `http://jellyfin.lab`
* Local URL: `http://server-ip:8096`
